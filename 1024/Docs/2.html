GraphLC IDE Special 1024发布
October 20, 2020
  • By Return

新版本特色功能

内嵌Chrome浏览器，支持网页预览，让Html开发更加方便
加入Python/CppCop以及Git模块

本次更新的内容

1.修复若干bug
2.加入Html、Python、Git Module
3.界面优化
4.模块修复
5.加入GraphLC IDE Command指令
6.内嵌Chrome浏览器内核，支持Html预览

注意，由于GraphLC IDE Special 1024 与GraphLC IDE Offcial Ver 2020共用同一个Module资源，因此新的Module在GraphLC IDE Offcial Ver 2020中不可用，您需要重新安装更新。

重要(2.2020.5.1024以下版本)

由于在GraphLC IDE Offcial Ver 2020中发现更新漏洞，因此您需要手动更新IDE，当运行IDE时，可能会弹出“是否安装更新”等字样，请点击否。随后访问我们的官方站点来进行更新。
